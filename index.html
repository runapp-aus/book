Welcome

Welcome to R for Australian Public Policy, the book collaboratively authored by the R Users Network for Australian Public Policy.

This book aims to provide advice and guidance on using R in Australian public policy, including available open datasets and how to use them.

This book is also being written in the open, and you are welcome to contribute to it.

License

This work, as a whole, is licensed under the Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International license.

Other Resources

This book assumes that users have a base level familiarity with the R programming language. If you want to learn more about R and Data Science, the R for Data Science book is a great place to start.
